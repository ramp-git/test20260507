# 旬菜と炭火 こはる PRサイト

個人飲食店を想定した、静的なPR用Webサイトです。店舗のコンセプト、メニュー、店内紹介、アクセス、予約問い合わせ導線を1ページにまとめています。

## ファイル構成

- `index.html` — ページ構造と日本語コピー
- `styles.css` — レスポンシブ対応のビジュアルデザイン
- `script.js` — モバイルナビゲーションとデモ予約フォームの挙動

## 確認方法

ブラウザで `index.html` を開くか、ローカルサーバーを起動して確認します。

```bash
python3 -m http.server 8000
```

その後、`http://localhost:8000` にアクセスしてください。
